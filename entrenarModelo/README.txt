El archivo datosEntrenarModelo.xlsx contiene los datos limpios utilizados para entrenar el modelo 3 
con el algoritmo de bosques aleatorios, el cual ha sido seleccionado como el mejor modelo. 
Las variables presentes en el archivo son las siguientes:

Fecha
DiaN
TMax
TMin
Mj/m2/d
Viento
PVA
ETPF56

El objetivo del archivo trainModel.py es entrenar un modelo de bosque aleatorio utilizando estos datos y 
luego guardar el modelo resultante en formato .sav para su uso posterior.
